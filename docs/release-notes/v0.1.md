# v0.1 — multiqc-foundation-gate

First public release. **Capability portrait, not a research result.**

## What this version proves

A complete substrate-bracketed analytical pipeline — ENA SRR download
through FastQC + MultiQC, into a 28-dim feature vector, through three
cross-validated classifiers, and out to drift detection — runs in a
single Python process and emits a hash-chained audit ledger that covers
every stage.

The substrate is the deliverable. The classifier ranking on this n=50
demo is *intentionally* one where the simplest model wins:

| Method               | Accuracy (5-fold CV) | Macro F1 |
|----------------------|----------------------|----------|
| LogisticRegression   | **0.860**            | 0.847    |
| RandomForest         | 0.840                | 0.823    |
| MLP (PyTorch, ~1.5k params) | 0.400 (collapse on majority class) | 0.190 |

The MLP collapse is documented in the README as a *feature of honest
scoping*, not a bug to hide: 1.5k parameters on 50 samples with 3
classes will under-fit even with strong regularization. The substrate
framing (per-epoch audit + MLflow + comparison artifact) is identical
to what a production n=3,000 MLP would emit.

## What ships

- **Substrate (4 channels)**: audit, tracking, canary, drift —
  hash-chained NDJSON ledger, MLflow no-op fallback, daily canary probe.
- **Data layer**: 10 Himes (SRP033351) base SRR + 4 synthetic augmentation
  strategies = 50 MultiQC reports, all committed as classifier-input
  fixtures.
- **Feature layer**: 28-dim vector (7 numerical + 10 module status + 3
  aggregate + 8 presence flags) — pipeline-agnostic, shared across all
  five augmentation strategies.
- **Classifier layer**: PyTorch MLP + sklearn RandomForest + sklearn
  LogisticRegression, all on identical StratifiedKFold splits, with
  per-fold audit emissions and side-by-side `comparison.json`.
- **Drift layer**: KS 2-sample test per feature against include-class
  baseline, with constant-column edge case handling, top-5 drifted
  feature surfacing, and `drift.json` artifact.

## Reproducibility

- `make run` end-to-end on n=50: **3.76 s wall-clock** on a Mac M-series.
- **49 unit tests** across model / train / baseline / eval / drift /
  pipeline / labels / audit, all green.
- **215-entry** hash-chained audit ledger, verified by `audit.verify()`
  with tamper-detection coverage in the test suite.
- English-only CI scanner enforces a CJK-free public surface.

## Honest scope

- n=50 with 10 unique base SRR — not a research-grade dataset.
- Single Himes airway smooth-muscle cohort — no cross-pipeline transfer
  tested.
- Synthetic augmentation is deterministic (seed=42) and label-perfect —
  real adapter contamination / quality drift will look messier.
- Production deployment items (HA, RBAC, model registry beyond MLflow
  artifacts, drift-remediation policy) are explicitly out of scope —
  see `docs/what-is-out-of-scope.md`.

## What's next
